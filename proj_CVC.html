Crowd Analysis Computer Vision Pipeline

Overview

This project sought to investigate the topology of influence networks in crowds. Description of the experiment itself For this project I trained a number of object detection models (I tried just about every one of them to nail down a performant option) and then used the detections to track the movement of people in the scene. I then used the tracks to build a graph of the scene and used the graph to compute a number of metrics that describe the topology of the scene. This project was one of my first forays into machine learning and computer vision. Ironically, since this project has launched Meta has shipped a number of open source CV models which can accomplish the goals of this pipeline, but I'm still proud of this and it was a hell of a lot of fun to figure out.
